# DigiTools

DigiTools is a modern digital product marketplace web application where users can explore premium digital tools, add products to cart, and manage selected items in a clean and responsive interface.

## 🔗 Live Website
[Visit DigiTools](https://digitool-app.netlify.app/)


## Project Description

DigiTools is built to provide a smooth and modern shopping experience for digital products such as premium tools, templates, and productivity resources. Users can browse available products, add them to their cart, remove them, and view total pricing dynamically.

The project focuses on reusable React components, conditional rendering, state management, and responsive UI design.


## Technologies Used

- Tailwind CSS
- DaisyUI
- React Icons
- React Toastify
- JavaScript (ES6)
- JavaScript XML
- React


##  Features

### 1. Add to Cart Functionality
Users can add products to the cart, and selected products are instantly reflected in the cart section.

### 2. Dynamic Cart Management
Users can remove items from the cart and see the total price update automatically.

### 3. Responsive & Modern UI
The website is fully responsive and designed with a clean modern layout using Tailwind CSS and DaisyUI.
